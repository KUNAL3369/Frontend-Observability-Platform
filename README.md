## 🧠 The Problem It Solves

Frontend apps **silently degrade** — slow load times, API latency spikes, and rising error rates go undetected until users are already frustrated.

This platform gives engineering teams the visibility to act first:

- ⚡ Monitor frontend performance in **real time**
- 🚨 Detect anomalies **before users report them**
- 📉 Understand trends across **devices and time ranges**
- 🔬 Simulate how **internal observability tooling** works at scale

## 🔥 Key Highlights

- Real-time simulated data engine
- Alert system with threshold logic
- Role-based UI (Admin / Manager / Analyst)
- Global filters affecting entire app state

## ✨ Features

| Feature                      | Description                                       |
| ---------------------------- | ------------------------------------------------- |
| 📊 **Real-Time Metrics**     | Live-updating page load, API latency & error rate |
| 🚨 **Alerts System**         | Threshold-based alerts with a notification panel  |
| 🔐 **Authentication**        | Supabase-powered login with protected routes      |
| 🛡️ **Role-Based Access**     | Admin / Manager / Analyst with scoped permissions |
| 🔎 **Global Filters**        | Filter by date range and device type              |
| 📈 **Interactive Charts**    | Recharts-powered trend graphs and breakdowns      |
| 📂 **Multi-Page Dashboards** | Overview → Performance → Errors                   |
| 💾 **Export**                | Download metrics as JSON                          |

## 🏗️ How It Works

```
User Login (Supabase Auth)
        │
        ▼
Role Detection → Admin / Manager / Analyst
        │
        ▼
Protected Routes (React Router)
        │
        ├── Overview      →  Key metrics at a glance
        ├── Performance   →  Trends & device breakdown
        └── Errors        →  Alert history & anomaly tracking
                │
                ▼
        Global Filters (date range, device type)
                │
                ▼
        Real-Time Data Engine (continuous updates + history)
                │
                ▼
        Alert Engine → fires when thresholds are crossed
```

**Role permissions:**

| Role       | Capabilities                              |
| ---------- | ----------------------------------------- |
| 🔴 Admin   | Full control — reset metrics, export data |
| 🟡 Manager | Limited actions                           |
| 🟢 Analyst | Read-only                                 |
